# 원고 상태

이 문서는 `/home/windmorning/projects/SIPWorks/SIPLite/docs/book` 원고의 현재 상태와 남은 작업을 정리한 편집용 메모다.

## 현재 상태

- 장 수: 36개 파일 수준
- 범위:
  - 서문/맺음말
  - 구조 장
  - SIP 흐름 장
  - TLS/transport 장
  - 설정/운영/배포 장
  - 테스트/로그/부록 장
  - 보안/성능/리팩터링 장
  - 출판용 목차/그림/표 초안
- 성격:
  - 코드 분석서로는 충분히 읽을 수 있는 수준
  - 출판용 원고로는 편집 중반 단계를 지난 초안 수준

## 이미 완료된 항목

1. 프로젝트 전체 구조 분석
2. REGISTER, INVITE, ACK/BYE/CANCEL, SUBSCRIBE/NOTIFY 흐름 정리
3. Digest 인증, XML 설정, TLS 구현 분석
4. 로그, 테스트, 운영 체크리스트 정리
5. 리팩터링, 보안, 성능 관점 분석
6. 서문, 맺음말, 최종 요약 추가
7. 출판용 목차 초안 추가
8. 그림 초안과 표 초안 추가
9. 핵심 장 일부에 실제 그림/표 본문 삽입
10. 장 끝 제목 형식을 `이 장의 핵심 정리`로 통일

## 지금 원고의 강점

- 코드 기준 설명이어서 근거가 분명하다.
- 파일 경로와 함수 참조가 풍부하다.
- SIP 개념, 구현, 운영, 리스크가 모두 연결되어 있다.
- 책으로 확장할 수 있는 뼈대가 이미 갖춰져 있다.

## 지금 원고의 약점

- 장마다 문체와 밀도가 약간 다르다.
- 중복 설명이 아직 남아 있다.
- 그림과 표가 전 장에 실제 삽입된 것은 아니다.
- 출판본 기준으로는 용어 설명과 문장 압축이 조금 더 필요하다.

## 남은 권장 작업

### 1단계. 본문 편집

- 중복 문단 압축
- 다음 장 예고 문장 추가

### 2단계. 시각 자료 반영

- `34_figure_drafts.md` 내용을 본문 장에 순차 삽입
- `35_table_drafts.md` 내용을 본문 장에 순차 삽입
- 필요 시 Mermaid와 ASCII 중 하나로 통일

### 3단계. 출판용 정리

- `33_publication_toc_draft.md` 기준으로 부/장 재배치
- 부록 문체 압축
- 서문과 맺음말을 출판용 톤으로 다듬기
- 장 번호를 출판본 기준으로 재정렬

### 4단계. 검증

- 코드 변경과 문서 불일치 점검
- 실제 테스트/실행 로그와 문서 내용 대조
- TLS, 운영 체크리스트, 보안 한계 표현 재검토
